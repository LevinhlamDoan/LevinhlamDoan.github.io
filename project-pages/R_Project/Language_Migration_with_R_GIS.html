Data Analysis Portfolio

Exploring GIS and R for Data Analysis

Summary

This portfolio showcases projects leveraging data from the UK Census 2021, focusing on migration flow, language proficiency, and employment statistics. By analyzing these datasets, I aim to uncover insights about the socio-economic landscape, exploring how language skills impact employment and how migration patterns shape local economies.

1. Spatial Analysis of Language Proficiency and Economic Activity

This project analyzes the spatial distribution of language proficiency and spatial migration, using R and GIS techniques.

2. Geocoding and Mapping Economic Activity by MSOA

Creating a detailed map showing economic activity levels across different MSOAs, visualizing data with R's `ggplot2` and spatial packages.

3. Time Series Analysis of Economic Changes

Investigating temporal changes in language proficiency and economic activity, utilizing historical data to uncover trends over time.

4. Interactive GIS Dashboard with Shiny

Developing an interactive dashboard to explore the relationship between language proficiency and economic activity across MSOAs using R and Shiny.

5. Cluster Analysis of MSOAs

Performing cluster analysis to identify regions with similar characteristics in terms of language proficiency and economic activity, visualized on a map.

6. Correlation Analysis with Spatial Context

Investigating correlations between language proficiency and economic activity within a geographical framework, using heatmaps and geographic maps.

7. Visualizing Movement Patterns

Analyzing and visualizing movement patterns between MSOAs based on Linestring data, understanding impacts on language proficiency and employment.

8. Data Cleaning and GIS Integration

Documenting the data cleaning process and the integration of GIS components in R, highlighting the importance of data integrity and spatial context.
